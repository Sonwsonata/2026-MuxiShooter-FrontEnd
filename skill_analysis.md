# 局内技能分析

## 伤害公式
伤害公式=(10+攻击力)*(1+图鉴攻击加成+所有伤害增加效果-所有伤害降低效果+其他效果(如狂阶弹幕尺寸修正)*暴击时(1+100%+图鉴爆伤加成+其他爆伤加成)

## 技能系统设计要点

1. **伤害计算系统**: 需要实现复杂的伤害公式，包括攻击力、图鉴加成、伤害增减、暴击等
2. **技能冷却系统**: 每个技能有独立的CD，需要支持CD减少的强化
3. **弹幕系统**: 不同技能发射不同模式的弹幕（直线、横向、随机等）
4. **强化系统**: 每个技能有多个强化选项，分为金、银、铜三个等级
5. **Buff系统**: 某些技能提供临时增益效果（攻速、移速、暴击等）
6. **穿透系统**: 部分弹幕可以穿透敌人
7. **镜像系统**: AI工具技能需要生成镜像
